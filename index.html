Read each situation, then choose the correct third conditional sentence. Focus on the full structure — both the if-clause and the result clause must be right.

Third Conditional — Formula
If-clause: If + had + past participle
If they had spent … / If she hadn't gone …
Result clause: would/could/might + have + past participle
… he would have passed / … she could have done it
❌ Common errors: spended · could shopping · would not went · would not late · did not fixed
