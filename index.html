Third Conditional — Formula
If-clause: If + had (not) + past participle
If I had been there … / If she hadn't gone …
Result clause: would/could/might + have + past participle
… I would have helped / … she could have done it
❌ Common errors: had went · would helped · could shopping · had knew · would not late
How the quiz works — example
I wasn't there, so I couldn't help.
If I had been there, I would have helped you.
